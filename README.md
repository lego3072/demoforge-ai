# DemoForge AI

Build role-specific demo flows with ROI framing, objection handling, and handoff actions for agent or human sales teams.

## Run locally
```bash
pip install -r requirements.txt
uvicorn app.main:app --reload --port 8000
```

## Endpoints
- `/`
- `/docs-page`
- `/health`
- `/v1/public/config`
- `/llms.txt`
